(s)CSS Shapes

SCSS mixins of some of the most common shapes.

Examples

Rectangles
                    
  • #rectangle {
    • 
                    
  •     @include rectangle(100px, 40px, red);
    • 
                    
  • }
    •
    Parallelograms

    •

    • 
                    
  • }

  • }
